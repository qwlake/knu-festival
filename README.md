2. 술 친구 게시판
3. 푸드트럭
4. 강대총학 문의 게시판

# Environments
```
Windows or IOS
Python 3.6+
Django 2.1
```

# Quick Start
```
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
